Music Player Mod Alpha v0.1

Play your own custom songs inside Minecraft using a simple in-game music player GUI.

Current Features

  • Import custom .ogg songs
  • In-game music player GUI
  • Reload songs without restarting
  • Auto generated resourcepack
  • Simple song management system

Planned Updates

  • Volume control slider
  • Direct MP3 support
  • Loop button
  • Playlist system
  • Random song button
  • Better UI design

GUI Preview

Installation Guide

  • Put mod inside mods folder
  • Start Minecraft
  • Enable Songs Mod resourcepack
  • Type /instructions
  • Click Add Song
  • Add .ogg files
  • Press Reload Songs
  • Press F3 + T
